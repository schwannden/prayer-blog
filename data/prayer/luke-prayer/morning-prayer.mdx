# 禱告

> 主─以色列的神是應當稱頌的！因他眷顧他的百姓，為他們施行救贖， \
> 在他僕人大衛家中，為我們興起了拯救的角， \
> 正如主藉著從創世以來聖先知的口所說的話， \
> 拯救我們脫離仇敵和一切恨我們之人的手， \
> 向我們列祖施憐憫，記念他的聖約─ \
> 就是他對我們祖宗亞伯拉罕所起的誓─ \
> 叫我們既從仇敵手中被救出來， \
> 就可以終身在他面前，坦然無懼的用聖潔、公義事奉他。 \
> 孩子啊！你要稱為至高者的先知；因為你要行在主的前面，預備他的道路， \
> 叫他的百姓因罪得赦，就知道救恩。 \
> 因我們神憐憫的心腸，叫清晨的日光從高天臨到我們， \
> 要照亮坐在黑暗中死蔭裡的人，把我們的腳引到平安的路上。
>
> > 路1:68-79

## 默想

早晨我們默想的是上帝的救贖。可以在靈修前、靈修後、或者上班的路上默想。

救贖的行動中包含*救*與*贖*。*救*是上帝拯救的實際事件，從創世伊甸園開始，經過出埃及，到耶穌道成肉身，被釘十字架，以及復活。*贖*強調的是神將我們贖回，並與我們同在。上帝的拯救並非冷漠的行動，而是與上帝的同在（以馬內利）不可分割。有時候我們的困境並未改變，但我們仍然感謝主，因為神讓我們更深地經歷祂的同在。

上帝的救贖源於祂的約，因為這是祂所立的約，我們可以有保障。這份安全感讓我們歌唱：“眾山怎樣圍繞耶路撒冷，耶和華也照樣圍繞他的百姓，從今時直到永遠。”

救贖要解決的首要問題不是我們眼前的困境，而是我們與神隔絕的窘境。因此，救贖的結果是讓我們能夠終身在祂面前，坦然無懼地用聖潔、公義事奉祂。

記得我們是蒙恩典、蒙拯救的，因此調整今天我們面對家人、面對同事的態度，記得我們所走的是一條平安的路，也要把人帶上這條路，與主同行。
